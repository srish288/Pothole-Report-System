Pothole Reporting System
Description
A web-based pothole reporting system that allows users to submit road damage reports with location details and images. The backend handles data storage and provides an admin dashboard to view and manage submitted reports.
Features
⦁	Users can submit pothole reports with description, location, and photo
⦁	Image upload handled using Multer
⦁	Reports stored in JSON format on the server
⦁	Admin dashboard to view and delete reports
⦁	REST APIs for report submission and retrieval
Technologies Used
⦁	JavaScript
⦁	Node.js
⦁	Express.js
⦁	Multer (file uploads)
⦁	HTML/CSS (basic admin interface)
